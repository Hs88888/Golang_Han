## 3. 了解Golang核心概念
### GOROOT和GOPATH 区别
- GOROOT是go的工程目录
- GOPATH是go的安装目录
###### Tips：无论把项目放在哪里，最好创建一个go文件夹，然后在 go/ 下再创建一个 src 的子文件夹

### 三个目录
| --bin (存放编译后的可执行文件)
| --pkg (存放编译后的包文件)
| --src (存放项目源文件)
